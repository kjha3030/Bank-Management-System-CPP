# Bank Management System (C++)

A console-based Bank Management System developed using C++.

## Features

- Create Account
- Deposit Money
- Withdraw Money
- Check Balance
- View All Accounts
- Search Account
- Delete Account
- File Handling

## Technologies Used

- C++
- File Handling
- GCC Compiler

## How to Run

```bash
g++ bank.cpp -o bank
./bank
```
